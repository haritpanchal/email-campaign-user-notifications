The Email Campaign User Notifications plugin lets users send emails to selected users with selected roles.

== Description ==
The plugin\'s primary goal is to send campaign emails to all the users on the gap of a particular amount of time not all at once to reduce the website load.  

Users can select different settings in the plugin tabs like,
1. Select the number of users to send mail at once. (If selected 5 then the plugin will send 5-5 users an email notification)
2. Select the time gap at which every email will be fired. ( If selected 60 then the plugin will send an email notification to (e.g) 5 users after every 60 seconds.
3. Users can add custom email subjects and content in the plugin.
4. Test the template via email in the \'Test Email\' tab.
5. Select roles and on based on that, users will receive email notifications.
